# Feihong's PureScript Quickstart

## Prerequisites

    yarn global add purescript pulp psc-package

## Create new project

    cd <project-directory>
    pulp --psc-package init

## Commands

Install dependencies

    psc-package install

Build and run 

    pulp run

Build

    pulp build

Run tests

    pulp test
